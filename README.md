## Unikernels, or: Docker without the hype

- Setup Rump Kernel (NetBSD Anykernel) toolchain ([Wiki](https://github.com/rumpkernel/wiki))
- Take a look at `app.cc` and `make.sh`
- Run `./make.sh [toolchain] [platform]`
